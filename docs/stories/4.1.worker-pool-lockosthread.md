# Story
**As a** 性能可靠性负责人
**I want** 将并发模型设计为 vCPU=worker 并 `LockOSThread`
**so that** 稳态 CPU < 75%，队列深度 < 1，稳定低延迟。

# Acceptance Criteria
1. N=Enclave vCPU（基线 12）时，稳态 CPU < 75%，`engine_q_depth_avg < 1`
2. 每 worker 固定签名上下文，`LockOSThread()` 绑定线程
3. 使用 MPSC 队列，队满快速失败（不阻塞热路径）
4. 可观测：`engine_q_depth`, `worker_busy_ratio`
